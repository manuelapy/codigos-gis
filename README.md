oie, tudo bem?

aqui eu vou disponibilizando aos poucos alguns códigos que vêm facilitando meu dia a dia no trabalho.

IMPORTANTE !! Estes códigos não são para uso no terminal e sim para serem salvos como um script na caixa de ferramentas:

1) Abra caixa de ferramentas
2) Parte superior > encontre símbolo do python > Adicionar script à caixa de ferramentas.
3) Seleciona o script baixado.
4) Depois ele fica salvo na caixa de ferramentas e é só usar quando quiser.
